== Description ==

Supercharge your WooCommerce checkout! Seamlessly introduce an irresistible order bump at the final step, delighting your customers with must-have extras right before they seal the deal.

Imagine this: Your customers are moments away from completing their purchase, and just before they click that 'Buy Now' button, they're presented with irresistible offers and must-have products they can't resist. With our plugin, you can effortlessly add an order bump to your WooCommerce checkout page, turning each transaction into an opportunity to boost your revenue.

It's never been this simple to create a checkout experience that leaves your customers delighted and coming back for more.

= Main Features =

- Seamless integration into your WooCommerce store
- Easily customize your order bumps to match your brand
- Tailor your order bumps and product offerings
- Effortlessly promote additional products
- Maximize your revenue potential
- Responsive design, mobile ready layout
- Live preview
- Simple and variable product support
- Multi-step checkout support

Three simple steps:

1. Pick a product to offer
2. Set a promotional price and enter a catchy copy
3. Add an image and choose a design

✅ That's it! Seat tight and watch as your average order value soars, all while enhancing your customer's shopping experience.

== Installation ==

= Automatic Plugin Installation =

To install this plugin using the built-in plugin installer:

1. Go to: Plugins > Add New.
2. Type in **Molongui Order Bump for WooCommerce** in Search box.
3. Click on **Install Now** to install the plugin.
4. The resulting installation screen will list the installation as successful or note any problems during the install.
5. If successful, click **Activate Plugin** to activate it, or **Return to Plugin Installer** for further actions.

= Manual Plugin Installation =

To install a WordPress plugin manually:

1. Download the plugin to your computer.
2. Extract the plugin folder to your desktop.
3. With your FTP program, upload the extracted plugin folder to the **wp-content/plugins/** folder in your WordPress directory.
5. Go to **Plugins** screen and find the newly uploaded plugin in the list.
6. Click on **Activate** link to activate it.

If you need further assistance, please, read the [Installing Plugins](https://wordpress.org/support/article/managing-plugins/#installing-plugins "Installing Plugins") article on the WordPress Support site or write us an email so we can assist you.

= Requirements =

In order to run, Molongui Order Bump for WooCommerce requires:

* PHP 5.3.0 or greater. Considering that "[WordPress recommends](https://wordpress.org/about/requirements/ "WordPress requirements")" it to be 7.4 or greater, this should not be a problem.
* WordPress 5.2.0 or later.
* WooCommerce 2.5.0 or later.

That's really it. We recommend using PHP 7.4 and the latest versions of WordPress and WooCommerce.


== Frequently Asked Questions ==

= Which e-commerce plugins is Molongui Order Bump for WooCommerce compatible with? =

**Molongui Order Bump for WooCommerce** is fully compatible with WooCommerce, version 2.5.0 or greater.

= Is the plugin compatible with WooCommerce? =

Yes, version 2.5.0 or greater.

= Is the plugin compatible with multi-step checkout page plugins? =

Yes, as long as those multi-step checkout page plugins keep in place WooCommerce hooks.

= Can I have more than one deal? =

Yes! For that you need to purchase Molongui Order Bump for WooCommerce Pro. Check it out [here](https://www.molongui.com/deals-sales-promotions-and-upsells-for-woocommerce/ "Get Molongui Order Bump for WooCommerce Pro").

= Can I configure the deal to be displayed only when there is a specific product in the Cart? =

Yes! For that you need to purchase Molongui Order Bump for WooCommerce Pro. Check it out [here](https://www.molongui.com/deals-sales-promotions-and-upsells-for-woocommerce/ "Get Molongui Order Bump for WooCommerce Pro").

= I need to run a scheduled promotion. How can I configure that? =

For that you need the "scheduling" option that is available only on the Pro version of the plugin. Check it out [here](https://www.molongui.com/deals-sales-promotions-and-upsells-for-woocommerce/ "Get Molongui Order Bump for WooCommerce Pro").

= Is it possible to have the deal checked by default? =

Yes! You can configure the deal to be checked by default, which means that the offered product will be automatically added to your client's Cart without any action required. That is a premium feature, though. You can check it out [here](https://www.molongui.com/deals-sales-promotions-and-upsells-for-woocommerce/ "Get Molongui Order Bump for WooCommerce Pro").
